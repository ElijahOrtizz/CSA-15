# CSA-15 Crypto Scalping Indicator

## Overview

CSA-15 is a TradingView Pine Script indicator designed for high-frequency crypto scalping.
It uses multi-confirmation signals to generate high-probability BUY/SELL alerts.

##  Features

* Multi-indicator confirmation system
* Real-time BUY/SELL alerts
* Designed for 15-minute scalping strategy
* Compatible with Webull execution via TradersPost

##  Strategy Logic

The indicator combines:

* EMA trend direction
* Momentum confirmation
* Volume signals
* Candle structure

##  Tech Stack

* TradingView Pine Script
* Webull API (via TradersPost)

##  Future Improvements

* AI-based signal filtering
* Backtesting dashboard
* Risk management automation
